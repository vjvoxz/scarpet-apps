# Changelog

All notable changes to the Effective Netherite Armor Scarpet app will be documented in this file.

Changes follow the format: **MAJOR.MINOR.PATCH**  
- **MAJOR**: Breaking changes or major new features.  
- **MINOR**: Backwards-compatible feature additions.  
- **PATCH**: Bug fixes or small improvements.  

The latest changes appear at the top.

---
## [v1.1.0] - 2026-07-24
### Added
- Logic to cleanly remove the infinite Fire Resistance effect when armor is removed.
- Safeguard to prevent stripping normal Fire Resistance potions.
### Changed
- Switched to using an infinite duration (-1) for the Fire Resistance effect.

## [v1.0.0] - 2026-07-24
### Added
- Initial release of the Effective Netherite Armor Scarpet app.
- Grants continuous Fire Resistance to players wearing a full set of Netherite armor.
